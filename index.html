Summary of the paper:

- Optimization Case Study (using Python)

This repository contains a capaciated lot sizing problem of a door sub-assembly lines in a worldwide home appliance maker's refrigerator factory. The main topic is to determine lot sizes and production sequence for the factory's mixed-models assembly line. A mathematical formulation is discussed and a python script for Gurobi solver (an optimization software) is also uploaded.
